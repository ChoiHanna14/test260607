# 🤖 김비서 - Smart Subscription & Task Management Platform

## 📋 프로젝트 개요

**김비서**는 구독 서비스, 업무, 매출 데이터를 한눈에 관리할 수 있는 통합 대시보드 플랫폼입니다. 
은행 연동 없이 데이터를 기기에 저장하고, AI 기반 분석으로 효율적인 업무 관리를 지원합니다.

---

## ✨ 주요 기능

### 📊 대시보드
- **월간 지출 현황**: 실시간 구독료 추적
- **활성 구독 관리**: 카테고리별 필터링 (9가지 분류)
- **예정 결제 알림**: D-day 형식으로 다음 결제일 표시
- **절약 팁**: 월간 절약액 자동 계산
- **캘린더 뷰**: 월별 결제 일정 시각화

### 🎯 업무 관리
- **할 일 목록**: 우선순위별 정렬 (높음/보통/낮음)
- **진행 상태 추적**: 진행중/대기/완료 상태 관리
- **팀별 역할 분담**: 담당자별 업무 할당
- **마감일 관리**: 우선순위 기반 작업 정렬

### 📈 성과 분석
- **프로젝트 진행률**: 프로그레스 바로 시각화 (6개 프로젝트)
- **매출 데이터**: 월별 추이 및 제품별 비교
- **차트 분석**: 선 그래프(월별 추이) + 막대(제품별 비교)
- **팀 성과**: 담당자별 업무량 및 진행상황

### 📝 데이터 분석
- **회의록**: 체계적인 회의 기록 및 액션 아이템 관리
- **프로세스 다이어그램**: 업무 흐름 시각화 (기획→제작→검토→배포→분석)
- **사이트 분석**: 경쟁사 분석 및 개선사항 정리

### 🎨 사용자 경험
- **다크/라이트 모드**: localStorage로 설정 저장
- **글래스모피즘 디자인**: 프리미엄 UI/UX
- **반응형 레이아웃**: 모바일 완벽 지원
- **상단 네비게이션**: 페이지 간 빠른 이동

│   └── 회의록.txt            # 마케팅팀 주간회의 기록
│
└── 📂 정리된 문서 (정리해줘/)
    ├── 📋 보고서/            # 보고서 버전별 정리
    ├── 📝 메모/              # 아이디어 및 피드백
    ├── ✅ 업무/              # 진행중인 작업 기록
    └── 📦 기타/              # 미분류 임시 파일

## 🛠️ 사용 기술

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - Flexbox & Grid 레이아웃
  - 글래스모피즘 (backdrop-filter)
  - 그라디언트 배경
  - CSS 변수로 테마 관리
- **JavaScript (Vanilla)**:
  - 동적 데이터 렌더링
  - 로컬스토리지 (localStorage)
  - Canvas를 이용한 차트 그리기
  - DOM 조작 및 이벤트 처리

### 디자인 & UX
- **반응형 디자인**: @media queries
- **다크/라이트 모드**: CSS 변수 활용
- **글래스모피즘**: 투명도 + 블러 효과
- **프리미엄 색상**: 파스텔/파란색/초록색 계열

### 데이터 형식
- **CSV**: 구조화된 데이터 (매출, 프로젝트, 업무)
- **TXT**: 텍스트 기반 정보 (일정, 회의록)
- **SVG**: 벡터 기반 프로세스 다이어그램

### 배포 & 호스팅
- **GitHub**: 소스 코드 관리
- **Vercel**: 자동 배포 (GitHub 연동)
- **Git**: 버전 관리 및 협업
