# 실시간 번역앱 기획안

> 작성일: 2026-06-14  
> 플랫폼: iOS (iPhone)  
> 목표: 일본 소셜/커뮤니티 앱을 사용하면서 앱을 떠나지 않고 실시간으로 한국어 번역

---

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 실시간 번역앱 |
| 플랫폼 | iOS (iPhone) |
| 번역 방향 | 일본어 → 한국어 |
| 번역 엔진 | Google Gemini API (텍스트 + 이미지 OCR 동시 처리) |
| 핵심 방식 | iOS Share Sheet Extension (앱을 떠나지 않고 번역) |

---

## 2. 핵심 동작 방식

### 공유 버튼 번역 (메인 기능)

일본 앱을 사용하는 도중, 앱을 나가지 않고 아래에서 번역 슬라이드가 올라오는 방식

```
일본 앱 사용 중
    ↓
번역하고 싶은 텍스트 또는 이미지 길게 누름
    ↓
iOS 공유 버튼 탭
    ↓
"실시간 번역앱" 선택
    ↓
화면 아래에서 번역 결과 슬라이드 올라옴
(뒤에 일본 앱 화면이 그대로 보임)
    ↓
[복사] 또는 [닫기] 선택
    ↓
다시 일본 앱으로 복귀
```

### 화면 구조 (Share Sheet 활성화 시)

```
┌─────────────────────────┐
│                         │
│     일본 앱 화면        │  ← 뒤에 그대로 보임

│                         │
│                         │

│        良くて散歩した。 │
│                         │
│   [클립보드 복사] [닫기] │
└─────────────────────────┘
```

## 3. 번역 대상 콘텐츠

| 콘텐츠 유형 | 지원 여부 | 처리 방식 |
|------------|----------|----------|
| 사용자 게시글 / 댓글 | ✅ | 텍스트 직접 번역 |
| DM / 채팅 메시지 | ✅ | 텍스트 직접 번역 |
| UI 텍스트 (버튼, 메뉴) | ✅ | 텍스트 직접 번역 |
| 이미지 속 일본어 | ✅ | Gemini Vision OCR 후 번역 |

---

## 4. 기능 목록

### 4-1. 핵심 기능

- **Share Sheet 번역**: 텍스트 / 이미지 공유 → 한국어 번역 슬라이드
- **탭-번역**: 스크린샷 위에서 원하는 영역 탭 → 해당 블록 전체 번역
- **클립보드 복사**: 번역 결과를 한 번에 복사
- **원문 + 번역 동시 표시**: 반투명 한국어 텍스트가 원문 위에 오버레이

### 4-2. 보조 기능

- **번역 히스토리**: 지금까지 번역한 내용 목록으로 보관
- **스크린샷 업로드 번역**: 이미지 파일 직접 불러와서 번역
- **설정**: API 키 입력, 번역 표시 방식 선택

## 6. 기술 스택

| 항목 | 기술 |
|------|------|
| 앱 빌드 방식 | PWA (Progressive Web App) + iOS Share Extension |
| 프론트엔드 | HTML / CSS / JavaScript (기존 프로젝트와 동일) |
| 번역 API | Google Gemini API (OCR + 번역 동시 처리) |
| API 보안 | Vercel Serverless Function (GEMINI_API_KEY 서버에서 관리) |
| 데이터 저장 | localStorage (번역 히스토리) |
| 배포 | Vercel (기존 프로젝트에 통합) |

## 9. 사용자 시나리오 예시

**시나리오: 일본 트위터에서 게시글 번역**

1. 트위터 재팬 앱에서 일본어 트윗 발견
2. 트윗 텍스트 길게 누름 → "공유" 탭
3. 공유 메뉴에서 "실시간 번역앱" 선택
4. 아래에서 슬라이드 올라옴: "오늘 라멘 먹었는데 진짜 맛있었어요 🍜"
5. [복사] 탭해서 클립보드에 저장
6. 슬라이드 닫고 다시 트위터로 복귀

**시나리오: 일본 앱 이미지 속 텍스트 번역**

1. 일본어 텍스트가 담긴 이미지 발견
2. 이미지 저장 후 실시간 번역앱에서 불러오기
3. 번역하고 싶은 부분 탭
4. Claude Vision이 이미지에서 일본어 인식 → 한국어 번역 표시
